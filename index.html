Project Link

This project was a dive into all things graphics programming implementing a range of techniques such as shadow mapping, physically based rendering, and GPU based terrain generation. Using OpenGL as a low level basis for the project.

This project was a university assignment where I was tasked with making a small slice of a game using a light weight graphics libaray(SFML) and the rest of the code base being fully implemented from scratch using C++. The project applies many C++ concepts such as template programming and RAII along with having many features such as spatial partitioning, multi-threading, and A-star pathfinding.
