Project responsibilites

Leaderboard

Leaderboard that updates across clients based on a centralised server object that tracks player information throughout the match. Also, displays player tag as highlighted so players know who is who on the leaderboard allowing them to strategise based on each other's points.

Tower Placement System

Surface based tower placement system that will snap and rotate towers to a surface based on its bounding box.

Towers

Programmed all tower mechanics in the game by taking a state machine approach to how towers behave, of which towers implement as an interface making it quick and easy to create new towers. All tower logic is also replicated across client and server using the authoritative architecture unreal provides.
