Project responsibilites

Procedural Generation

Programmed procedural generation system that would generate terrain chunks of different biomes as the player progressed. This also included unique generation of interactable objects and enemies per biome.This system was also optimised using object pooling.

Tutorial system

Tutorial system that would save player progress knowing which actions were successful.This ensured that the tutorial was reactive to the player and less repetitive on failure.

Event Manager

Programmed event generation managed by a centralised event object.This allowed for multiple in game events(such as different obstacles) to be randomly cycled during gameplay keeping things fresh and dynamic.

Collectables Manager

Programmed collectable generation based on single centralised manager object. Collectables generate in random patterns for each procedurally generated terrian chunk.
